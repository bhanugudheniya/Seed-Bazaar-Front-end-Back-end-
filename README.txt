Software Require:
------------------
1. Node.js : https://nodejs.org/en/
2. MongoDB : https://docs.mongodb.com/manual/installation/
3. Angular : https://angular.io/cli
4. NPM : https://docs.npmjs.com/downloading-and-installing-node-js-and-npm

*****************************************************************************************

Steps:
-------

1. First, clone this repository.
2. Then, open "seed-bazaar" and install npm in terminal using - npm install.
3. Next, run project using "ng serve --open" in terminal.
4. Open another directory " seedbazaarbackend" and install npm in terminal using - npm install.
5. Next, run backend using "npm start" or "nodemon start" in terminal.
6. Open fresh CMD and write command - "mongod".
